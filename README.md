# Blockchain explorer for Cosmos SDK

---

## Getting Started

1. `npm install`
2. Open `.env.development` and `.env.production`
3. Fill your API url address, site name, denom and minimal denom:
```text
VITE_API_URL=https://
VITE_SITE_NAME=""
VITE_DENOM=""
VITE_MINIMAL_DENOM=""
```
4. `npm run dev`
